# HandMadeCandle-Ecommerce
Candle Web is a premium e-commerce frontend built with React 19, Vite, and TypeScript. It features a stunning responsive UI with a gold/cream aesthetic, product curation, real-time cart management, and secure user dashboards. Optimized for performance and user experience, it includes professional PDF order reports and seamless API integration.

## üöÄ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. **Clone the repository**:
  
2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Configure Environment Variables**:
   Create a `.env` file in the root of the Frontend directory and add your API URL:
   ```env
   VITE_API_URL=https://your-api-url.com
   ```

4. **Run the development server**:
   ```bash
   npm run dev
   ```

5. **Build for production**:
   ```bash
   npm run build
   ```
